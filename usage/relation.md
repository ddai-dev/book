## API

一个基本的好友关系功能应该具有以下操作：

``follow(my_id, target_id)`` 关注给定用户。

``unfollow(my_id, target_id)`` 取消对给定用户的关注。

``following(my_id)`` 返回所有被我关注的人。

``count_following(my_id)`` 返回我关注的人数。

``follower(my_id)`` 返回所有关注我的人。

``count_follower(my_id)`` 返回关注我的人数。
